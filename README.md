## 📝 Changelog

<details>
<summary><b>Версия 1.13: Modular Edition</b> (текущая)</summary>

- **Модульная архитектура** – код разделён на отдельные файлы (`core.js`, `vfs.js`, `themes.js`, `game_service.js`, `style.css`), что упрощает поддержку и расширение.
- **Система визуальных тем (пресетов)** – добавлено более 20 готовых тем в `themes.js` и `themes.css`, включая:
  - Windows 1.0 / 2.0 / 3.1 (Retro)
  - Windows 95 / 98 / 2000 (Classic)
  - Windows XP (Luna), Vista/7 (Aero), 8 (Metro), 10 (Fluent), 11 (Mica), 12 (Concept)
  - macOS, Ubuntu, ChromeOS, KDE Plasma, AmigaOS, MS-DOS Terminal
  - Каждая тема меняет CSS-класс, иконку меню «Пуск», фоновый цвет, набор переменных и положение элементов управления.
- **Игровой сервис (GameService)** – добавлен модуль `game_service.js` для сохранения рекордов и времени, проведённого в играх-приложениях.
- **Улучшенное прилипание окон (Snap)** – новая опция в Центре управления, позволяющая включить/отключить прилипание к краям экрана; анимация подсветки зон стала более плавной.
- **Сохранение размеров окон** – при изменении размера окна его ширина и высота запоминаются и восстанавливаются после перезагрузки системы.
- **Анимация Центра управления** – добавлены плавные анимации появления и исчезновения.
- **Умная панель задач** – панель корректно перестраивается при вертикальной/горизонтальной ориентации; иконки не выходят за границы экрана.
- **Исправления и оптимизация**:
  - Окна больше не теряют фокус при перетаскивании iframe.
  - Устранены рывки при перемещении окон.
  - Улучшена производительность отрисовки иконок.
  - Мелкие исправления в работе IndexedDB, вёрстке и обработке событий.

</details>
